# Frontend structure

```
src/
  App.jsx
  api.js
  main.jsx
  main.css
  components/
    common/          # reusable small components
    layout/          # navbar, sidebar, cart, notifications
  pages/
    auth/            # login and register
    student/         # student/customer pages
    owner/           # restaurant owner/staff pages
  admin/             # admin module, layout, pages, shared admin UI
  assets/            # images and logos
```
